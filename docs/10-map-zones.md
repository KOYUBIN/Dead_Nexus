## 개요

DEAD NEXUS의 무대는 **애시그리드(Ashgrid)** — **11 × 11 = 121개 구역**으로 구성된 도시 격자.  
중앙에 넥서스 타워, 그 주변에 **동심원 구조**로 업타운 → 미드타운 → 다운타운 → 비통제구역 접경이 펼쳐진다.

### v0.3 재설계 요점

| 항목 | v0.2 (6×6=36) | v0.3 (11×11=121) |
|---|---|---|
| 총 구역 | 36 | 121 (3.4배) |
| 구조 | 남북 분할 (업/미드/다운타운) | **동심원 5링** (중앙 NEXUS 기준) |
| 블록 시작 구역 | 2 | 3 |
| 플레이 시간 목표 | 120분 | **180분 (3시간)** |
| 이동 규칙 | 1포인트=1칸 | 1포인트=1칸 + **방사형 하이웨이** (2칸/포인트) |

### 왜 동심원인가

1. **세계관 일관성**: 넥서스 평의회 = 권력의 중심 → 블록은 중앙 가까이 포진, 고스트는 변두리에서 본다
2. **거리 = 권력 거리**: 블록 본사(ring 1) → 업타운(ring 2) → 전장(ring 3) → 슬럼(ring 4) → 비통제(ring 5)
3. **공평한 대칭**: 5개 블록이 중앙을 둘러싸고 서로를 견제 — 방위별 비대칭이 아닌 원형 대칭
4. **플레이 타임 확보**: 더 넓은 맵 → 3시간 플레이 체감 (이동·탐색 충분한 여유)

## 1. 맵 구조

### 1.1 격자 배치

11열(A~K) × 11행(1~11) = 121 구역. 중앙 F6에 NEXUS 타워.


좌표계: 알파벳(A~K, 서→동) × 숫자(1~11, 북→남). 중앙은 **F6**.

### 1.2 동심원 링 구조

중앙 F6에서의 **Chebyshev 거리**(∞-norm)로 링 결정:

| 링 | 거리 | 구역 수 | 테마 | 지배 세력 |
|---|---|---|---|---|
| **Ring 0** | 0 | 1 | NEXUS 타워 | 5대 블록 공동 |
| **Ring 1** | 1 | 8 | 블록 본사 벨트 | 5 Bloc HQ + 3 넥서스 부속 |
| **Ring 2** | 2 | 16 | 업타운 | 블록 직접 지배 (금융·데이터·고급) |
| **Ring 3** | 3 | 24 | 미드타운 | 블록·고스트 경계지대 (전장) |
| **Ring 4** | 4 | 32 | 다운타운 | 서민·고스트 활동 (슬럼·항구) |
| **Ring 5** | 5 | 40 | 비통제구역 접경 | 고스트 영토·무법지대 |
| **합계** | — | **121** | | |

검산: 1 + 8 + 16 + 24 + 32 + 40 = **121** ✓

### 1.3 인접성 규칙

- **상하좌우 4방향** 인접 (대각선 미포함)
- 같은 링 내 이동·인접 링으로의 이동 모두 1포인트/칸
- 맵 외곽(A, K, 1, 11행열) 밖은 **비통제구역** — 레거시 모드에서 접근 가능

## 2. 중앙 구역 — NEXUS (F6)

5대 블록 공동 이사회 타워. 도시의 상징.

- 게임 시작 시 **중립**
- 매 수익 단계: 모든 속성 각 1 + ₵ 3 + 인플루언스 1
- 기본 방어력 +3 (최강)
- 장악 조건: **2라운드 연속 점유** 필요
- **3라운드 연속 장악 = 즉시 승리**

## 10. 이동 규칙

### 10.1 기본 이동
- 인접 구역 간 1칸 = 1이동 포인트
- 이동 자원: ₵ 1 또는 연료셀 1 또는 SPD 판정
- 이동 포인트는 SPD 스탯에 비례 (기본 SPD = 1라운드당 이동 포인트)

### 10.2 방사형 하이웨이 (v0.3 신규)

11×11 맵의 광대함을 보완하는 **8방향 방사형 도로**.  
NEXUS(F6)에서 8방위로 뻗은 직선을 따라가는 이동은 **1포인트 = 2칸**.

**하이웨이 좌표**:
| 방위 | 경로 |
|---|---|
| 북 (N) | F1 → F2 → F3 → F4 → F5 → [F6 NEX] |
| 남 (S) | [F6] → F7 → F8 → F9 → F10 → F11 |
| 서 (W) | A6 → B6 → C6 → D6 → E6 → [F6] |
| 동 (E) | [F6] → G6 → H6 → I6 → J6 → K6 |
| 북서 (NW) | A1 → B2 → C3 → D4 → E5 → [F6] |
| 북동 (NE) | K1 → J2 → I3 → H4 → G5 → [F6] |
| 남서 (SW) | A11 → B10 → C9 → D8 → E7 → [F6] |
| 남동 (SE) | K11 → J10 → I9 → H8 → G7 → [F6] |

**하이웨이 규칙**:
- 하이웨이 cell에서 다음 하이웨이 cell(같은 축)로 이동: **1포인트 = 2칸 점프**
- 단, 경찰서(F3, F9, C6, I6)는 하이웨이상에 있지만 점령 불가 → 통과만 가능
- 하이웨이 통행 시 **공권력 트랙 상승 없음** (공공 도로)

### 10.3 특수 이동
- **DRIFTER**: 모든 이동 비용 0 (패시브)
- **지하 루트**: 항구 간 직접 연결 — F2↔F10, D10↔F10, H10↔F10, A6↔K6, F1↔F11 (3칸 점프)
- **메시다이브**: CIPHER 메시 순간 이동 (챕터 5 이후)
- **봉쇄 무시**: DRIFTER STORM RUSH 카드

### 10.4 플레이 타임 계산

| 시나리오 | 6×6 (v0.2) | 11×11 (v0.3) |
|---|---|---|
| 중심 → 모서리 (직선) | 5칸 (5포인트) | 10칸, 하이웨이 5포인트 |
| 중심 → 모서리 (대각선) | 5칸 | 10칸, 하이웨이 5포인트 |
| 평균 이동 거리 | 3~4칸 | 4~6칸 (하이웨이 적극 활용 시) |
| 라운드당 이동 소요 | 3~5분 | 5~8분 |
| 게임 총 시간 | 120분 | **180분 (목표)** |
