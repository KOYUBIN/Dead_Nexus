## 1. 첫 플레이 권장 흐름

### 솔로 (1인) 첫 플레이 — 30분
1. 시뮬레이터 열기 → **모드: 솔로** / **맵: 5×5 튜토리얼** / **난이도: EASY** / **역할: Ghost · BLADE**
2. R1 시작. 화면 좌측 플레이어 패널에서 본인 (P0 ★) 확인
3. Phase 2 (계획)에서 손패에서 카드 2장 선택 — TOP/BOT 골라 확정
4. 이동이 있으면 맵에서 목표 셀 클릭
5. 자동 진행 후 Phase 5 (수익) → Phase 6 (R&D) → 트랙 1개 무료 LV 업
6. 5R 이후 승리 조건 도달 시 종료
7. **첫 게임 목표**: 시스템 친숙해지기. 승리 못 해도 OK

### 핫시트 (2~3인) 첫 플레이 — 60~90분
1. **모드: 핫시트** → 인간 P1·P2(·P3) 역할 선택. 빈자리는 봇 보충
2. R1 시작. **PASS THE DEVICE** 화면이 매 인간 차례 시작 시 자동 표시
3. P1 차례: 손패 보고 카드 2장 선택 → "✓ 확정" 클릭
4. PASS DEVICE 화면 → P2에게 기기 전달 → P2 차례
5. 모든 인간 plan 끝나면 자동 Phase 3 진행
6. 봇과 인간 모두 동시에 카드 공개·실행
7. **첫 게임 목표**: 협상 1회 시도해보기 (자원 스왑 또는 truce 약속)

## 3. 첫 5라운드 단계별 가이드 (Ghost · BLADE)

### R1 (시장 / 뉴스 / 계획 / 실행)
- **시장**: 거래 패스 (자원 부족)
- **뉴스**: 카드 효과 확인 (보통 ±1~2 영향)
- **계획**: BASIC MOVE + QUICK DRAW 선택 → 인접 Bloc 구역으로 이동
- **실행**: Bloc 구역 도착 시 **레이드 결정 모달** → 폭력형 raid 시도 (성공 확률 ~50%)
- **수익**: 0~3 자원 획득
- **R&D**: 화력 트랙 LV3로

### R2 (공격 강화)
- **계획**: SUPPRESS_FIRE + DOUBLE_TAP → 두 번째 raid 시도
- **레이드**: 첫 raid 성공 시 🗡 첫 레이드 성공 하이라이트 ★+2
- **R&D**: 그리드런 LV2 (이동 +1)

### R3 (Cyberware 자동 장착)
- **R&D 종료 시 자동**: 리플렉스 부스터 ⚡ 장착 (ATK +1, HP -1)
- 이제 ATK 화력 LV2 (+1) + 화력 LV3 패시브 (+1) + 부스터 (+1) = ATK 3 + 4 = **7**
- 평판 누적 18~20 도달

### R4 (NEXUS 도전)
- 11×11이면 NEXUS F6로 이동 시도. 첫 도달 시 🌐 ★+3
- 5×5는 NEXUS C3 도달 시 동일

### R5 (승리 시도)
- **5×5**: 평판 42 + raid 2회 → 전투 승리 / 평판 60 단독 승리
- **11×11**: 평판 45 + raid 2회 → 전투 승리 / 평판 70 단독 승리
- 보통 R5~R7에 승리 결정

## 4. 자주 묻는 질문 (FAQ)

**Q. 카드 두 장 선택했는데 효과가 둘 다 발동되나?**
A. Yes. 한 장은 TOP/MAIN, 한 장은 BOT/SIDE 영역으로 이미 분리. 클릭으로 영역 전환 가능.

**Q. mini-raid가 뭔가요?**
A. atk 카드가 없는 클래스(MOLE/RIGGER/BLADE/CIPHER)가 카드 효과로 인접 Bloc을 자동 중립화하는 메커니즘. raid count도 누적됨.

**Q. 협상은 시뮬에서 어떻게 하나?**
A. Phase 1.5에서 봇이 자동으로 거래 제안 → 평가 후 수락/거절. 사람용 모달은 v1.4 이후 추가 예정.

**Q. NEXUS는 점령했는데 룰이 활성화 안 됨?**
A. NEXUS가 Bloc 소유여야 룰 발동. Ghost가 점유 시 비활성. 또는 NEXUS가 중립 상태(누구 소유 X)도 비활성.

**Q. Cyberware 단점이 너무 큰데?**
A. 트레이드오프 디자인. 대신 게임당 2개 자동 장착이라 누적 보너스 큼. R3 + R6 자원 미리 비축.

**Q. 내가 raid 안 했는데 내 평판이 ★+1 됨**
A. 협상 페이즈에서 truce 약속 만료 → 양측 ★+1 보상. 또는 봇 raid 성공 시 트랙 보너스.

**Q. 패배 — HP 0 됐는데 다시 부활?**
A. 그림자 LV5 도달 시 deathSave 능력 (1회 한정). 또는 HELIX의 CLONE_DECOY 카드 효과.

## 5. 핫시트 모드 진행 팁

### 시간 분배 (3인 90분 기준)
- R0 (설정): 5분
- R1: 10분 (학습 라운드)
- R2~R4: 각 12분
- R5~R7: 각 8분 (속도 붙음)
- 종료 + 회고: 10분

### 협상 페이즈에서 (Phase 1.5)
- 시스템이 봇 거래만 자동 처리. 사람 간에는 자유 토론
- "이번 R 너 안 치면, 다음 R에 자원 줄게" 같은 약속을 시스템 truce에 등록 (수동)
- 협상은 메타게임 — 다른 사람 의도 추측이 핵심

### 차례 전환 (PASS DEVICE)
- 가림막이 본인 손패 가려줌 → 다른 사람 못 봄
- 본인 차례 끝나고 "다음 차례" 클릭 → 가림막 자동 표시
- 다음 사람이 "▶ 차례 시작" 누르면 손패 보임
