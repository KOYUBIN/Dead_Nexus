## 1. 5트랙 + 거리명성 (Street Cred) 시스템 (v0.6.3 ~ v1.0)

각 플레이어는 5개 트랙을 보유, 각각 0~5 LV.

| 트랙 | 한글 | 아이콘 | 라이프패스 | 효과 영역 |
|---|---|---|---|---|
| combat | 화력 | 💥 | Solo (킬러) | ATK·레이드 파워 |
| transport | 그리드런 | 🌃 | Nomad | 이동·도주 |
| mfg | 코드 | 💾 | Netrunner/Rigger | 부품·무기·해킹 |
| party | 인맥 | 🤝 | Fixer | 크레딧·영향력·접선 |
| security | 그림자 | 🎭 | 첩보 | DEF·위장·은폐 |

**거리명성(Street Cred)** = 5트랙 합. Cyberpunk RPG 핵심 지표.

### 1.1 트랙 LV별 효과

| LV | 효과 |
|---|---|
| 1 | 기본 (시작 트랙) |
| 2 | 패시브 +1 (atk/spd/def/hack 등 해당 능력) |
| 3 | 특수 레이드 타입 해금 (security 3 = 잠입형 / 인맥 3 = 협상형 / 코드 4 = 드론형) |
| 4 | 패시브 +3 누적, 강화 효과 |
| 5 | once-ability 해금: 화력=critFailImmunity / 그리드런=teleport / 코드=droneFree / 인맥=reroll / 그림자=deathSave |

### 1.2 거리명성 마일스톤

거리명성 6 / 12 / 18 도달 시 자동 보상:
- **6**: Ghost ★+1 ₵+2 / Bloc ₵+3 ⚙+1
- **12**: TL Progress +3 + 자원 약간
- **18**: Ghost ★+3 / Bloc ₵+5 영향력+1 + 🏆 전설 하이라이트

### 1.3 클래스별 시작 트랙

| 클래스 | 화력 | 그리드런 | 코드 | 인맥 | 그림자 |

| BLADE | 2 | 1 | 0 | 0 | 1 |
| CIPHER | 0 | 1 | 1 | 0 | 1 |
| RIGGER | 1 | 1 | 2 | 0 | 0 |
| BROKER | 0 | 2 | 0 | 2 | 1 |
| DRIFTER | 1 | 2 | 1 | 0 | 0 |

## 2. 협상 시스템 (Above-the-Table) — v0.9.2

매 R Phase 1.5 에 협상 페이즈 발동.

### 2.1 거래 종류

| 종류 | 설명 |
|---|---|

BROKER 클래스만 매 R 협상 시 자동 ★+1 ₵+1 인맥 트랙 +1. **클래스 정체성**.

## 11. 게임 모드 (v1.0)

### 11.1 솔로 (1인)
- 1 인간 + 3 봇 (4인 기본)
- 난이도: easy / normal / hard
  - easy: 봇 자동 점령 50%, 인간 시작 자원 +5
  - normal: 봇 75~100%
  - hard: 봇 100% + 봇 시작 자원 +5

### 11.2 핫시트 (2~5인)
- 2~5명이 한 기기 공유, 차례대로 plan
- Plan phase에 차례 전환 시 "PASS THE DEVICE" 가림막 표시
- 빈 자리는 봇 자동 보충 (4인 기준)
- 협상은 사람 간에 자유 토론 + 거래 (시스템은 truce/swap 기록만)

### 11.3 봇만 시뮬 (개발용)
- 헤드리스 sim-harness/run.js로 100~200판 자동
- 균형 검증 / 클래스 승률 측정
