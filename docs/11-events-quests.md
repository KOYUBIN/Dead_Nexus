**문서 ID**: `docs/11-events-quests.md`

**관련 문서**: `02-core-rules.md`, `10-map-zones.md`, `12-legacy-campaign.md`

## 개요

애시그리드의 각 구역에는 **이벤트 토큰**이 배치되어 있다.  
플레이어가 구역에 처음 진입하면 토큰이 공개되며, 네 가지 유형의 결과가 발생한다.

이 시스템은 게임에 **탐험의 재미**와 **예측 불가능성**을 부여하며, 퀘스트는 Ghost의 주된 수익원이 된다.

---

## 1. 이벤트 토큰 배치

게임 시작 시 맵의 각 구역에 이벤트 토큰 1개가 **뒤집힌 상태**로 놓인다.

### 토큰 구성 (맵 사이즈별)

**표준 11×11 (총 121개)** — Ring 0 NEXUS는 토큰 없음 → 실제 120개:

| 유형 | 비율 | 수량 | 색상 |
|---|---|---|---|
| 🔴 즉발 이벤트 (Instant Event) | 40% | 48개 | 빨강 |
| 🟡 퀘스트 (Quest) | 35% | 42개 | 노랑 |
| 🟢 발견 (Discovery) | 15% | 18개 | 초록 |
| ⚪ 공백 (Empty) | 10% | 12개 | 흰색 |

**튜토리얼 5×5 (총 24개)** — 중앙 C3 NEXUS 제외:

| 유형 | 비율 | 수량 |
|---|---|---|
| 🔴 즉발 이벤트 | 30% | 8개 |
| 🟡 퀘스트 | 40% | 10개 |
| 🟢 발견 | 20% | 4개 |
| ⚪ 공백 | 10% | 2개 |

> 튜토리얼 비율은 퀘스트 우선. 복잡 이벤트(레거시 토큰·사이버사이코 등) 제외.

### 최초 진입 규칙
- 플레이어가 구역에 **처음 도착**하면 이벤트 토큰 즉시 공개
- 효과 적용 후 토큰 제거
- 다른 플레이어는 해당 구역 재진입 시 이미 공개된 정보로 취급

### 토큰 사전 확인
다음 능력으로 토큰을 미리 확인 가능:
- **RIGGER 드론 정찰**: 인접 구역 1칸 토큰 확인
- **AXIOM 서베일런스 넷**: 최대 3칸 동시 확인
- **CIPHER 포트 스캔**: 현재 구역 + 인접 토큰 확인

## 3. 퀘스트 (Quests)

35% 비율, 13개. 공개 시 해당 퀘스트 카드를 플레이어의 **퀘스트 슬롯**에 배치.

### 퀘스트 슬롯
- 플레이어당 최대 **2개** 동시 보유
- 슬롯 꽉 차면 새 퀘스트 거부 또는 기존 퀘스트 포기
- BROKER 클래스는 슬롯 3개 (패시브)

### 퀘스트 유형 (8종)

| 유형 | 목표 | 보상 |
|---|---|---|
| **배달 (Delivery)** | 자원 X를 지정 구역 Y까지 운반 | ₵ 4 + 렙 2 |
| **암살 (Hit)** | 지정 고스트·NPC 제거 | 렙 4 + 무기 2 |
| **해킹 (Breach)** | 구역 베일 해제 (HACK 판정) | 데이터 3 + 렙 2 |
| **협상 (Deal)** | 두 블록 간 계약 성사 중개 | ₵ 6 + 접선 2 |
| **탐사 (Explore)** | 미개척 구역 5칸 방문 | 부품 3 + 레거시 토큰 |
| **레이드 (Raid)** | 블록 구역 습격 성공 | 렙 5 + 블록 주가 −3 |
| **생존 (Survival)** | 전투 3회 연속 생존 | HP 최대 +2 (영구) |
| **밀수 (Smuggle)** | 무기를 압박 없이 지정 구역까지 | ₵ 5 + 연료 2 |

### 퀘스트 시간 제한
각 퀘스트는 **3라운드 제한**을 가진다.
- 제한 내 완료: 정상 보상
- 초과 시: 퀘스트 실패 → 렙 −2, 카드 파기
- 레거시 모드: 실패 기록 영구 남음

### 퀘스트 의뢰인
퀘스트는 의뢰인에 따라 구분된다:

| 의뢰인 | 퀘스트 성격 | 추가 보상 |
|---|---|---|
| 블록 (VANTA 등) | 블록 이익 관련 | 해당 블록 지분 +5% |
| 거리 | 고스트끼리의 의뢰 | 렙 +1 추가 |
| 비통제구역 | 애시그리드 외부 | 희귀 자원 |
| 익명 | 정체불명의 의뢰 | 보상 불명확, 위험 높음 |

## 4. 발견 (Discovery)

15% 비율, 5개. 즉시 긍정적 효과.

### 발견 목록

| # | 발견 | 효과 |
|---|---|---|
| 1 | 비밀 저장고 | 크레딧 8 획득 |

## 5. 공백 (Empty)

10% 비율, 4개. 효과 없음.  
플레이어가 헛걸음을 한 것으로 간주되며, 해당 턴 행동 기회만 소비한다.

- 단, **탐사 퀘스트**의 "방문 구역 수"에는 카운트됨
- **DRIFTER 고속 이동** 카드 사용 시 공백 구역 통과에도 패널티 없음

## 7. 이벤트 시스템 전략

### Ghost의 관점
- **즉발 이벤트**는 리스크. 감당 가능한지 먼저 판단
- **퀘스트**는 주 수입원. 적극적으로 탐사해 습득
- **레거시 토큰** 수집 우선 (챕터 해금)

### Bloc의 관점
- 구역 장악 전 이벤트 토큰 **사전 확인** 필수 (RIGGER 고용 or AXIOM TL 3)
- 내부자 정보로 주가 변동 예측 활용
- 부정적 이벤트 피하면서 긍정적 이벤트만 수집

### 탐사 페이스
- 초반 5라운드: 맵의 절반 이상 공개 권장
- 중반 6-8라운드: 핵심 구역 장악 + 퀘스트 수행
- 후반 9-12라운드: 남은 구역 정리 + 승리 조건 달성

- 퀘스트 50%

## 9. 레거시 모드의 이벤트

레거시 캠페인 진행 시:
- **소진된 이벤트는 재생성 안 됨** (일부 챕터 예외)
- **새로운 이벤트 토큰이 봉투에서 추가됨** (챕터 해금 시)
- **특수 이벤트**: 특정 캐릭터와 관련된 이벤트는 해당 플레이어에게만 발동
