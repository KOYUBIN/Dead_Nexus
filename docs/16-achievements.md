## 개요

DEAD NEXUS의 업적은 **두 층**으로 구성:

1. **🏆 싱글게임 업적 (In-Game Achievements)** — 한 판 안에서 달성하는 공개 도전
2. **🎖 메타 업적 (Meta Achievements)** — 여러 판에 걸쳐 누적·수집하는 훈장/타이틀

**목적**:
- 매 판마다 **부수적 동기** 제공 (승패 외에도 플레이할 이유)
- **장기 리플레이성** — 다음 판을 돌리게 만드는 수집/진척 시스템
- 레거시 캠페인과 **독립**적으로 작동 (기본 게임 + 반복 플레이용)

**원칙**:
- 승리 조건 변경 X (기본 승리는 그대로)
- 업적은 **누적 보너스** + **타이틀 + 명예**
- 메타는 **플레이어 프로필**에 기록 (Notion DB 또는 종이 시트)

---

## 🏆 싱글게임 업적 (In-Game)

### 특징
- **1판 내부**에서 달성
- **공개 상태** — 달성 즉시 선언 가능 (숨은 목표와 반대)
- 각 업적은 **게임당 1회** 수령 (중복 X)
- **누구나** 달성 가능 (역할 무관하지만 일부는 Bloc/Ghost 한정)
- 달성 시 **소소한 보너스** + **메타 업적 포인트**

### 보너스 구조
| 등급 | 보너스 | 메타 포인트 |
|---|---|---|
| 🥉 브론즈 (쉬움, 20장) | 자원 +3 / 렙 +2 | 1점 |
| 🥈 실버 (보통, 10장) | 자원 +5 / 렙 +5 | 2점 |
| 🥇 골드 (어려움, 5장) | 자원/렙 +10 | 5점 |

### 사용 방식
1. 게임 시작 시 싱글게임 업적 카드 **35장 전체 덱**을 테이블 중앙에 펼침 (또는 참조 시트 1매 출력)
2. 플레이어가 조건 달성 시 **선언 + 해당 카드 수령**
3. 같은 업적 두 번 달성 시 **먼저 선언한 사람 우선**
4. 게임 종료 시 수령한 카드 개수·등급 기록

상세 카드 35장: `cards/achievements/in-game.md`

## 🎖 메타 업적 (Meta)

### 특징
- **게임 횟수 누적** (평생 또는 프로필 단위)
- 다음 판을 할 **이유** 제공
- 달성 조건은 **여러 판에 걸친 조건**
- **타이틀** 수여 — 다른 플레이어에게 자랑·표시 가능
- **시각적 수집** (스티커·배지)

### 카테고리 5종

| 카테고리 | 설명 | 카드 수 |
|---|---|---|
| 🎓 **클래스 마스터리** | 각 Ghost/Bloc 다회 플레이·승리 누적 | 8 |
| 🏆 **승리 트로피** | 각 승리 조건·시나리오별 승리 | 6 |
| ⚡ **극한 도전** | 특이/어려운 조건 달성 | 6 |
| 📚 **컬렉션** | 전 카드 사용·전 시나리오 완주 | 3 |
| 💀 **어두운 훈장** | 패배·실수·불운 기록 (유머) | 2 |

**합계 25장**

상세 카드 25장: `cards/achievements/meta.md`

### 추적 방법

#### 옵션 A — Notion 플레이어 프로필
- Notion DB에 플레이어별 행 1개
- 각 메타 업적을 체크박스 속성으로
- 진행률 자동 계산 (`X/25 획득`)
- 다른 플레이어 프로필과 비교 가능

#### 옵션 B — 종이 체크리스트
- 프린트킷의 "메타 프로필 시트" 1매
- 달성 시 스티커 부착 또는 체크
- 비공식 기록 — 본인 관리

#### 옵션 C — 레거시 박스 동봉
- 게임 박스 내부에 전용 페이지
- 여러 세션 걸쳐 연필로 기록
- 박스가 "이 그룹의 기록"

## 싱글 + 메타 연결

### 포인트 적립
- 싱글게임 업적 달성 시 해당 등급만큼 **메타 포인트** 적립
- 일부 메타 업적은 **포인트 누적**으로 해금
  - 예: "100포인트 달성" → 메타 업적 "DEDICATED PLAYER"
  - 예: "골드 업적 20회" → 메타 업적 "LEGEND"

### 연쇄 해금
일부 싱글 업적은 메타 업적 해금의 **필수 조건**:

| 싱글 업적 | → 연관 메타 업적 |
|---|---|
| "무전투 승리" (싱글 골드) | "BLOODLESS" (메타) — 3회 누적 시 획득 |
| "1라운드 사망" (싱글 브론즈) | "HOT START" (메타, 유머) — 1회로 충분 |
| "모든 블록 공격" (싱글 실버) | "EQUAL OPPORTUNITY" (메타) — 5회 |

## 시나리오 × 업적 상호작용

### S01 표준
- 모든 업적 활성

### S02 코프 대전
- Ghost 한정 업적 **비활성**
- "M&A 3회 성공" 업적 난이도 **쉬움**으로 재분류
- 전용 업적 "CORPORATE WARRIOR" 활성화 (S02 승리 필요)

### S03 스트리트 라이징
- Bloc 대형 업적 대부분 **비활성** (1명뿐)
- Ghost 업적 완화 (렙 기준 −20%)

### S04 계엄의 밤
- "공권력 10 도달" 관련 업적 **자동 달성** (이미 7에서 시작)
- "무전투 승리" 난이도 **어려움으로 +1**

### S05 골드러시
- 경제 업적 달성 보너스 +50%
- "자산 100 달성" 특수 시나리오 업적 활성

### S06 마켓 크래시
- "주가 0 → 재건" 관련 업적 활성
- "경제 재건자" 전용 메타 업적 해금 가능

## 프로필 시스템 (Notion 연동)

### 플레이어 프로필 페이지 구성

각 플레이어 Notion 페이지에 자동 집계:

- **이름·닉네임·시작일**
- **플레이 세션 수** (세션 로그 DB 관계)
- **싱글 업적 획득 목록** (누적)
- **메타 업적 획득** (25장 체크리스트)
- **선호 클래스·블록** (세션 로그 집계)
- **승패 기록**
- **특별 순간 기록**

### 리더보드 (선택)
- 메타 포인트 합계 기준
- 클래스별 승률
- "가장 많이 사용된 시나리오"
- "가장 큰 자산 기록"
- "가장 긴 생존 라운드" (Ghost)

## 설계 원칙

1. **승패와 분리** — 업적 때문에 승리 포기 유인 X. 달성 시 소소한 보너스만.
2. **모든 역할 참여 가능** — 역할 전용 업적 있지만 최소 1개는 모든 역할이 달성 가능
3. **싱글은 공개, 메타는 개인** — 싱글은 경쟁 유발, 메타는 개인 자랑
4. **어두운 훈장** — 실패·실수도 업적화. 보드게임 특유의 "그때 웃겼던 순간"을 수집
5. **수집의 즐거움** — 메타 25장이 완주 가능한 범위. 너무 많으면 포기, 너무 적으면 재미 없음

## 도입 단계

### Phase 1 — 입문 (Session 00~02)
- **업적 시스템 OFF**
- 기본 규칙 익히기 집중

### Phase 2 — 중급 (Session 03~)
- **싱글 업적만 ON**
- 메타는 아직 OFF
- 매 판 부수 목표 경험

### Phase 3 — 고급 (5판 이상 경험자)
- **싱글 + 메타 전부 ON**
- 프로필 시트 시작
- 장기 수집 동기 부여
