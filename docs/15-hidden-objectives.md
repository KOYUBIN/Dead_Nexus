# 15. 숨은 목표 시스템 (Hidden Objectives)

**문서 ID**: `docs/15-hidden-objectives.md`

**최종 수정**: 2026-04-22  
**관련 문서**: `02-core-rules.md`, `14-scenarios.md`, `16-achievements.md`

## 개요

각 플레이어는 공식 승리 조건 외에 **2장의 비밀 목표 카드**를 받는다. 게임 종료 시 달성한 목표만큼 **자산·렙·특수 타이틀** 보너스. 어떤 목표는 아예 **대체 승리 조건**이다.

**왜 필요한가**:
- 공식 승리 조건만 있으면 상대의 움직임이 예측 가능 → 메타가 고착
- 비밀 목표는 **왜 저 사람이 저걸 하지?** 미스터리 생성
- 엔드게임에서 **서프라이즈 역전** 가능
- 같은 역할(예: VANTA 2번째 플레이)도 목표가 다르면 완전히 다른 경험

---

## 핵심 규칙

### 1. 배부
- 게임 시작 시 각 플레이어에게 **비밀리에** 2장 배부
- Bloc 플레이어 → Bloc 목표 덱에서 2장
- Ghost 플레이어 → Ghost 목표 덱에서 2장

### 2. 유지·폐기
- 2장 중 **1장 유지 + 1장 폐기** OR **둘 다 유지**
  - 기본 모드: 둘 다 유지 (2개 병행)
  - 집중 모드: 1개만 유지 (선택의 무게 ↑)
- 폐기한 카드는 버림 (공개 안 함)

### 3. 공개 타이밍
- 기본: 게임 종료 시 동시 공개
- 옵션: 달성 즉시 선택적 공개 (보너스 즉시 수령)

### 4. 달성 판정
- 게임 종료 시 각 카드의 조건 체크
- 달성 시 카드에 명시된 보너스 수령
- 미달성 시 효과 없음 (페널티 X)

### 5. 충돌
- 같은 목표를 여러 플레이어가 뽑을 수 있음 (덱 크기 20장이므로 4~5인에서 중복 드물지만 가능)
- 공동 달성 가능 (상호 배타적인 목표만 예외 명시)

---

## 목표 유형 3종

### 🏆 달성형 (Achievement-type)
특정 조건 달성 시 **자산/렙 보너스**. 기본 승리 조건과 병행.

예: *"게임 종료 시 자산 40+ 도달"* → 자산 +10 보너스

### 🏁 대체 승리 (Alternate Victory)
기본 승리 조건을 달성하지 못해도 이 조건 달성 시 **독자적 승리**.

예: *"라운드 10 종료 시 HELIX 주가 3 이하"* → 이 플레이어 즉시 승리 (대역전)

### ⚖ 상호 배타 (Mutual Exclusive)
두 목표가 정반대. 한 플레이어가 달성하면 다른 플레이어는 자동 실패.

예: VANTA는 *"AXIOM 파산 유도"*, AXIOM은 *"VANTA 파산 유도"*

---

## 보너스 수령 방식

### 자산 보너스 (Bloc)
- 게임 종료 시 자산 계산에 추가
- 예: +10, +15, +20, +30

### 렙 보너스 (Ghost)
- 게임 종료 시 렙 계산에 추가
- 예: +5, +10, +15

### 타이틀 (Both)
- 공식 승자 외에 **특수 타이틀** 획득
- 메타 업적 카드 1장 영구 부착
- 자산·렙 계산에 반영 안 되지만 기록용

### 대체 승리
- 즉시 게임 승리 (기본 승리 조건 무시)
- 선언 시 모두 공개

## 숨은 목표 × 업적 조합

숨은 목표 달성과 **싱글 게임 업적** (`docs/16`)을 동시에 달성하면 **더블 리워드**:

| 상황 | 보너스 |
|---|---|
| 숨은 목표 + 싱글 업적 동시 | 자산/렙 추가 +5 |
| 숨은 목표 2장 모두 달성 | 메타 업적 "DOUBLE AGENT" 획득 |
| 대체 승리 달성 | 메타 업적 "SHADOW WINNER" 획득 |

## 설계 원칙

1. **숨은 목표는 공식 승리 조건을 대체하지 않음** (기본 모드). 대체 승리는 어려움 등급만.
2. **페널티 없음**. 달성 못 해도 0. 시도의 동기만 제공.
3. **역할 유인(incentive) 강화**. Bloc 목표는 Bloc 플레이를 더 Bloc스럽게, Ghost는 더 Ghost스럽게.
4. **다른 플레이어 추측 가능하게**. 너무 은밀하면 재미 없음. 중간 이상의 플레이는 눈에 띔.
5. **스토리·세계관 반영**. VOSS 가문 밀약 같은 레거시 떡밥도 여기 녹임.

## 플레이 팁

### Bloc 플레이어
- 2장 중 1장은 공식 승리와 **시너지**, 다른 1장은 **대안**으로 쓰기
- 경쟁 블록이 같은 목표 뽑았을 가능성 체크
- 상대의 "이상한 움직임"을 목표 단서로 해석

### Ghost 플레이어
- 카르마 목표는 플레이 스타일 크게 바꿈 — 뽑는 순간 노선 결정
- 복수 목표는 BLADE·MOLE에 유리
- 공개 타이밍이 전략적 — 숨기면 카운터 당할 확률 ↓
