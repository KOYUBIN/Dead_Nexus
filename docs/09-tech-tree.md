## 개요

모든 플레이어(Bloc과 Ghost)는 **테크 레벨(TL) 1~5**를 가진다.  
TL이 올라갈수록 새로운 카드·능력이 해금되고, 스탯 상한이 증가한다.

**테크트리 종류**:
- 각 Bloc마다 고유 테크트리 (5 × 5 = 25개 능력)
- 각 Ghost 클래스마다 고유 스킬 트리 (6 × 5 = 30개 능력)

---

## 1. 테크 업그레이드 비용

TL 상승에는 자원 투입 + 대기 시간이 필요하다.

### 비용표

| 레벨 | 필요 자원 | 대기 라운드 |
|---|---|---|
| 1 → 2 | 부품 3 + ₵ 4 | 즉시 발효 |
| 2 → 3 | 부품 5 + ₵ 8 + 데이터 2 | 1라운드 후 |
| 3 → 4 | 부품 8 + ₵ 15 + 데이터 4 + 인플루언스 2 | 2라운드 후 |
| 4 → 5 | 부품 12 + ₵ 25 + 데이터 6 + 인플루언스 5 | 3라운드 후 |

### 대기 규칙
- 업그레이드 선언 시 자원 즉시 소모
- 대기 라운드 카운터가 0 도달하면 해금 완료
- 대기 중 다른 업그레이드 동시 진행 불가
- 업그레이드 취소 시 자원 50% 환불

### 긴급 업그레이드
- **RIGGER의 OVERCLOCK** 카드: TL 1단계 임시 상승 (이번 라운드 한정)
- 긴급 업그레이드는 대기 시간 없음, 단 일회성

## 2. TL별 해금 혜택 (공통)

모든 플레이어에게 공통 적용되는 TL 혜택:

| TL | 공통 혜택 |
|---|---|
| **TL 1** | 기본 덱 사용 가능 |
| **TL 2** | 스탯 상한 +1 (1개 스탯 선택) |
| **TL 3** | 카드 2장 추가 덱에 포함 가능 |
| **TL 4** | 스탯 상한 +2 (2개 스탯 선택) |
| **TL 5** | Artifact 등급 카드 1장 해금 (레거시 필수) |

## 3. Bloc 테크트리 상세

### 3.1 VANTA

**전문**: 보안·정보 지배  
**주 속성**: ◈M MESH + ◈S SHADE

| TL | 해금 능력 | 설명 |
|---|---|---|
| 1 | **베일 방어막 (Veil Shield)** | 사이퍼 해킹 1회 무효화. 라운드당 1회 발동. |
| 2 | **블랙사이트 (Black Site)** | 데이터 +3/라운드 생산 구역 건설. 외부 접근 불가. 최대 1개. |
| 3 | **코어텍스 와이어 (Cortex Wire)** | 아군 고스트 사망 시 1회 부활. 게임당 2회. |
| 4 | **섀도우넷 (Shadow Net)** | 상대 주식 거래 내역 실시간 열람. 패시브. |
| 5 | **제로 프로토콜 (Zero Protocol)** | 1회: 타 블록 주식 전량 강제 매도. 게임당 1회 |

### 3.2 IRONWALL

**전문**: 군사·중화기  
**주 속성**: ◈I IRON + ◈A ASH

| TL | 해금 능력 | 설명 |
|---|---|---|
| 1 | **표준 화기 (Standard Arms)** | 충돌 시 무기 비용 −1. 패시브. |
| 2 | **전술 드론 (Tactical Drone)** | 원거리 구역 감시. 이동 없이 정보 획득. 라운드당 2구역. |
| 3 | **장갑 APC (Armored APC)** | 레이드 방어 시 무기 비용 면제. 패시브. |
| 4 | **전술 AI (Tactical AI)** | 아군 충돌 시 상대 주사위 1개 제거. 패시브. |
| 5 | **궤도 타격 (Orbital Strike)** | 1회: 구역 1칸 즉시 초토화, 3라운드 수입 차단 |

### 3.3 HELIX

**전문**: 생체공학·스플라이스  
**주 속성**: ◈B BIO + ◈V VOLT

| TL | 해금 능력 | 설명 |
|---|---|---|
| 1 | **기초 스플라이스 (Basic Splice)** | 소속 고스트 HP +2. 패시브. |
| 2 | **신경 인터페이스 (Neural Interface)** | 고스트 액션카드 1장 추가 사용 허용 (라운드당). |
| 3 | **생체 교란 (Bio Disruption)** | 상대 고스트 행동 1회 무효화. 라운드당 1회. |
| 4 | **클론 배치 (Clone Deploy)** | 사망한 고스트 복제본 즉시 생성. HP는 원본의 50%. |
| 5 | **풀 스플라이스 (Full Splice)** | 소속 고스트 전원 클래스 제한 해제. 모든 속성 카드 사용 가능 |

### 3.4 AXIOM

**전문**: AI·스마트 시스템  
**주 속성**: ◈M MESH + ◈V VOLT

| TL | 해금 능력 | 설명 |
|---|---|---|
| 1 | **스마트링크 (Smart Link)** | 무기 명중 자동 계산, 주사위 리롤 1회. 라운드당 1회. |
| 2 | **암호화 코어 (Encrypted Core)** | 자신의 행동 카드 엿보기 완전 차단. 패시브. |
| 3 | **예측 알고리즘 (Prediction Algorithm)** | 뉴스 카드 다음 장 미리 확인. 매 라운드. |
| 4 | **자율 경비망 (Auto-Security Net)** | 구역 자동 방어. 행동 소비 없이 방어. |
| 5 | **카오스 프로토콜 (Chaos Protocol)** | 1회: 전체 플레이어 행동 카드 무작위 섞기 |

### 3.5 CARBON

**전문**: 에너지·자원 인프라  
**주 속성**: ◈V VOLT + ◈A ASH

| TL | 해금 능력 | 설명 |
|---|---|---|
| 1 | **정제 플랜트 (Refinery Plant)** | 크레딧 수입 구역 +1 추가. 패시브. |
| 2 | **합성연료 (Synth Fuel)** | 고스트 이동 비용 전액 지원 가능 (동맹 형성). |
| 3 | **에너지 그리드 (Energy Grid)** | 인접 구역 3개 수입 동시 획득. 매 라운드. |
| 4 | **독점 파이프라인 (Monopoly Pipeline)** | 특정 자원 구역 타 블록 접근 차단. 라운드당 1개. |
| 5 | **다크 에너지 (Dark Energy)** | 1회: 전체 크레딧 수입 차단 (1라운드). 게임당 1회 |

## 4. Ghost 클래스 스킬 트리

각 Ghost 클래스도 TL에 따라 새로운 스킬이 해금된다.

### 4.1 CIPHER (정보전·해킹)

| TL | 해금 스킬 | 설명 |
|---|---|---|
| 1 | **포트 스캔 (Port Scan)** | 구역 베일 강도 확인 |
| 2 | **스니프 패킷 (Sniff Packet)** | 메시 통신 1건 가로채기 |
| 3 | **데이터 크래시 (Data Crash)** | 코프 주식 −2 즉시 발동 (데이터 4 소비) |
| 4 | **유령 프로토콜 (Ghost Protocol)** | 이번 라운드 행동 카드 추적 불가 |
| 5 | **신 시티 재킹 (Nexus Jack)** | 전체 코프 주식 트랙 1칸씩 하락 |

### 4.2 BLADE (전투·집행)

| TL | 해금 스킬 | 설명 |
|---|---|---|
| 1 | **빠른 반응 (Quick Response)** | 이니셔티브 동점 시 자동 선공 |
| 2 | **더블 탭 (Double Tap)** | 공격 성공 시 한 번 더 굴림 (추가 피해) |
| 3 | **방패 파괴 (Shield Break)** | DEF 무시 공격 (라운드당 1회) |
| 4 | **라스트 맨 (Last Man Standing)** | HP 3 이하 시 ATK +3 |
| 5 | **반격의 춤 (Counter Dance)** | 모든 공격을 반격으로 전환 (이번 라운드) |

### 4.3 RIGGER (기술·제작)

## 5. 테크트리 전략 가이드

### Bloc 전략

각 Bloc의 TL 5 능력은 **게임 체인저**이지만 도달하기까지 시간과 자원이 많이 든다.

| Bloc | TL 우선순위 | 추천 속도 |
|---|---|---|
| VANTA | TL 2 (블랙사이트) → TL 4 (섀도우넷) | 중간 속도 |
| IRONWALL | TL 3 (장갑 APC) → TL 5 (궤도 타격) | 빠르게 |
| HELIX | TL 2 (신경 인터페이스) → TL 4 (클론 배치) | 중간 속도 |
| AXIOM | TL 1-2 빠르게 → TL 5 최우선 | 빠르게 |
| CARBON | TL 3 (에너지 그리드) 최우선 | 느리지만 장기적 |

### Ghost 전략

Ghost는 Bloc보다 TL 상승이 느리다 (자원 부족). 대신 클래스별 특화 스킬이 강력하다.

- **CIPHER**: TL 3 (데이터 크래시)가 최우선 — 주가 조작 가능
- **BLADE**: TL 4 (라스트 맨) 조기 확보로 생존력 극대화
- **RIGGER**: TL 2-3 빠르게 돌고 장비 제작으로 버티기
- **BROKER**: TL 3 (위기 협상)만 있어도 게임 흐름 주도
- **DRIFTER**: TL 4 (피없는 길) 필수 — 봉쇄 무시
- **MOLE**: TL 5 (쿠데타) 목표로 한우물 파기

## 6. 레거시 모드 테크트리 변화

레거시 캠페인 진행 중 다음 변화 발생:
- **챕터 클리어**: 플레이어가 자신의 TL 트리 중 1개 능력을 **영구 해금**
- **M&A로 흡수**: 흡수한 블록의 TL 1개 스킬을 자신의 트리에 추가 가능
- **캐릭터 사망**: 해당 Ghost의 TL 진행 기록 삭제 (영구 제외 모드)
