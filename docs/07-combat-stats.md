## 변경 요약 (v0.3)

- **레이드 명시 결정 (Ghost → Bloc 구역)**: 이동으로 Bloc 구역에 도착하면 자동 레이드가 아니라 사용자가 [레이드/회피] 선택. 성공률 = (7 − threshold + atk)/6 공개.
  - 성공: 렙+3, 해당 블록 주가-3, 구역 중립화, 수배+1
  - 실패: HP-3, 수배+1, 공권력+1
- **Ghost vs Ghost PvP**: 같은 구역 조우 시 결투
  - 양측 d6 + (공격 ATK vs 방어 DEF) → 2점 이상 차이: 승자 렙+2 + 패자 HP-2 + ₵최대 3 탈취
  - 2점 미만 차이: 무승부 (피해 없음)
- **구역 요새화**: `ZONE_FORTIFY` 카드로 구역에 `fortified` 플래그 누적 → 레이드 threshold +N

## 개요

DEAD NEXUS의 전투 시스템은 5가지 스탯으로 구성된다.  
각 스탯은 캐릭터의 행동 방식과 전투 역할을 결정한다.

---

## 1. 5가지 전투 스탯

| 스탯 | 기호 | 의미 | 주 사용처 |
|---|---|---|---|
| **HP** | ❤ | 체력 | 피해 흡수. 0 도달 시 전투 패배 |
| **ATK** | ⚔ | 공격력 | 충돌 시 주사위에 더함 |
| **DEF** | 🛡 | 방어력 | 상대 공격에서 차감 |
| **SPD** | ⚡ | 속도 | 이니셔티브 결정 |
| **HACK** | 💻 | 해킹력 | 메시 전투 · 베일 돌파 |

## 3. 전투 해결 순서

### STEP A — 전투 개시 조건
다음 중 하나 발생 시 전투 돌입:
1. 두 플레이어가 같은 구역을 점령 시도
2. Ghost가 Bloc 구역을 레이드
3. 플레이어가 다른 플레이어를 직접 공격 (습격·처형)
4. NPC와 조우 (갱단·공권력·블록 경비)

### STEP B — 이니셔티브 결정
양측 이니셔티브 수치를 비교:
- Ghost: 선택한 2카드 중 TOP 카드의 이니셔티브 숫자
- Bloc: 플레이한 카드 TL × 10

**낮은 숫자 = 먼저 행동**

**동점 시 처리 순서**:
1. Ghost 우선 (Bloc보다 먼저)
2. BLADE 클래스 자동 선공 (패시브)
3. SPD 높은 쪽
4. 가위바위보

### STEP C — 공격 선언
공격자:
- 기본 ATK 스탯
- + 사용 카드의 ATK 보너스
- + 무기 자원 1개당 +0.5 (소비 선택)
- + 속성 풀 보너스

방어자:
- 기본 DEF 스탯
- + 사용 카드의 DEF 보너스
- + 구역 방어력 (거점·요새화 포함)
- + 속성 풀 보너스

### STEP D — 주사위 판정
양측 각각 d6 1개 굴림.

**주사위 보너스 추가 조건**:
| 조건 | 보너스 |
|---|---|
| IRON 속성 풀 2 이상 | 공격측 주사위 +1 |
| MESH 속성 풀 2 이상 | 적 DEF −1 |
| BIO 속성 풀 2 이상 | 자신 HP +2 (전투 중 회복) |
| ASH 속성 풀 2 이상 | 구역 피해 +1 (부차 효과) |
| IRONWALL 패시브 | 공격측 주사위 +1 (항상) |

### STEP E — 최종 수치 계산
- 최종 공격값 = ATK + 주사위 + 보너스
- 최종 방어값 = DEF + 주사위 + 보너스
- **피해 = 최종 공격값 − 최종 방어값**
- 피해 ≤ 0: 빗맞음, HP 감소 없음
- 피해 > 0: 방어측 HP 감소

### STEP F — 결과 처리

**방어측 HP 0 도달**:
- 전투 패배
- 보유 자원 30% 약탈당함
- 시작 구역으로 이동
- 레거시 모드: 사망 기록 (완전 영구사망은 옵션 설정)

**공격측 점령 성공**:
- 구역 소유자 변경
- 해당 블록 주가 조정 (Bloc 구역 피탈 시 −1)
- 새 소유자 즉시 이벤트 토큰 공개

**상처 / 스캔들 카드 체크**:
- Ghost HP가 최대치의 50% 이하 도달 → **상처 카드 1장 덱 삽입**
- Bloc이 구역 피탈 당함 → **스캔들 카드 1장 덱 삽입**

## 4. Ghost vs Bloc 레이드 특례

Ghost가 Bloc 구역을 공격하는 경우의 특수 규칙.

### 방어 구성
Bloc 구역의 방어는 다음 요소로 구성:
- 구역 기본 DEF
- 설치된 베일 레벨 (HACK으로 돌파)
- 경비 무기 수치 (TL에 따라 증가)
- 거점 보너스 (FORWARD BASE 등)
- AXIOM 자율 경비망 (설치 시)

### 레이드 성공 시
- Ghost **렙 +3**
- 해당 블록 **주가 −3**
- 구역 내 자원 **50% 약탈**
- 블록 덱에 스캔들 카드 1장 삽입
- Ghost 현상수배 +1

### 레이드 실패 시
- Ghost **HP −3**
- Ghost 현상수배 **+1**
- 공권력 트랙 +1
- 해당 블록 주가 +1 (방어 성공 보너스)

## 5. 특수 전투 상황

### 5.1 3자 이상 충돌

같은 구역에 3명 이상 진입 시:
1. 이니셔티브 순서대로 2자 충돌을 연쇄 해결
2. 승자가 다음 도전자와 다시 전투
3. 최종 생존자가 구역 획득
4. 모두 중상 시 구역 "황폐 상태" (해당 라운드 아무도 점령 불가)

### 5.2 메시 전투 (Mesh Combat)

CIPHER가 메시다이브 카드로 메시에 진입 시:
- **HACK 스탯이 주 판정 수치** (ATK 대체)
- 물리 ATK·DEF 무효
- 베일 레벨이 핵심 방어 요소
- 양측 카드는 속성 기반 효과만 발동
- 실패 시 실제 HP도 **−2 (백래시)**

### 5.3 처형 시도 (계약 처형)

BLADE의 **CONTRACT KILL** 카드 사용 시:
- 표적과 BLADE의 SPD 판정
- BLADE SPD 승리: 전투 없이 표적 HP −∞ (즉사)
- 표적 SPD 승리: 일반 전투로 전환

### 5.4 부활 조건

레거시 모드에서 HP 0 도달 시:
- **HELIX CORTEX WIRE TL3 능력** 발동 가능 → 1회 부활
- **HELIX 의료구역 방문** → 부활 가능 (₵ 5 지불)
- **부활 없이 사망** → 캐릭터 영구 제외 (옵션에 따라)

## 6. HP 관리 & 회복

### 자연 회복
- **단판 모드**: 라운드 종료 시 HP 1 자연 회복
- **레거시 모드**: 자연 회복 없음. 반드시 치료 필요

### 인위적 회복 수단
| 방법 | 효과 | 비용 |
|---|---|---|
| 거점 휴식 | HP 3 회복 | 1턴 소비 |
| HELIX 의료구역 방문 | HP 완전 회복 | ₵ 3 |
| HELIX의 FIELD SPLICE 카드 | HP 최대의 50% 회복 | ◈B + ₵ 2 |
| Long Rest | HP 완전 회복 + 상처 카드 제거 | 챕터 종료 시 |
| BIO 속성 풀 연쇄 | HP +2 (1회) | 같은 턴 BIO 카드 2회 |

## 7. 상처 카드 (Wound Card) 상세

Ghost HP가 최대치의 **50% 이하**로 떨어지면 상처 카드 1장이 덱에 자동 삽입된다.

### 상처 카드 스펙
```
┌───────────────────────────┐
│  INITIATIVE : —           │  (이니셔티브 없음)
├───────────────────────────┤
│  ▲ TOP : STUNNED          │
│  이번 턴 TOP 행동 불가    │
├───────────────────────────┤
│  ▼ BOTTOM : BLEEDING      │
│  HP −1 추가 손실          │
└───────────────────────────┘
```

### 처리 방법
- 덱에서 상처 카드를 뽑으면 2장 슬롯 중 1장을 차지
- 해당 카드는 사용 불가 상태로 처리
- Long Rest 또는 HELIX 의료구역에서 제거 가능

---

## 8. 공권력 (Heat) 반영

Ghost가 전투에서 과한 행동을 하면 공권력 트랙 상승:
- 무기 사용 충돌: 공권력 +1
- 레이드 성공: 공권력 +1
- 공개 전투 (주택가 등): 공권력 +2

자세한 공권력 트랙 규칙은 `docs/02-core-rules.md` §6 참조.

## 9. 스탯 임시 상승·하락

카드나 이벤트 효과로 스탯이 임시 변동될 수 있다.

### 적용 규칙
- 같은 스탯에 여러 효과가 중첩되면 **합산** (스택)
- 상승·하락 효과는 **이번 라운드** 또는 **이번 전투**로 한정
- "영구" 표기된 효과는 세션 전체 또는 캠페인 전체 지속 (명시)

### 예시
- BLADE의 QUICK DRAW: ATK +3 (이번 전투)
- HELIX의 AUGMENTATION: 스탯 +2 (이번 세션 영구)
- RIGGER의 OVERCLOCK: 테크 레벨 +1 (이번 라운드)

## 10. 스탯 상한

스탯은 무한정 상승하지 않는다. 상한은 다음과 같다:

| 스탯 | 기본 상한 | TL 5 도달 시 상한 |
|---|---|---|
| HP | 클래스 기본값 × 2 | 클래스 기본값 × 2.5 |
| ATK | 10 | 12 |
| DEF | 8 | 10 |
| SPD | 8 | 10 |
| HACK | 8 | 10 |

---

## 11. 밸런스 원칙

### 클래스 간 밸런스
- 각 클래스 스탯 합계 17~22 범위 (5 차이)
- 고 HP 클래스는 저 SPD (BLADE: HP 10, SPD 3)
- 고 HACK 클래스는 저 HP (CIPHER: HP 6, HACK 5)
- MOLE·RIGGER 등 중간형은 유연성으로 보상

### 전투 밸런스
- 기본 전투는 **3~5턴**에 결판나도록 설계
- 소진 전략은 HELIX 등 일부 블록에만 허용
- BLADE 자동 선공은 강력하지만 HP 소모 빠름
