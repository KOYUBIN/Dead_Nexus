# DEAD NEXUS — 데드 넥서스

> *연결과 붕괴. 도시 중심부의 폐허 위에서 벌어지는 전쟁.*

**Dead Nexus**는 4인 전략 보드게임으로, 붕괴한 메가시티 **Ashgrid(애시그리드)** 를 무대로 벌어지는 자원 관리 · 구역 장악 · 세력 전쟁 게임입니다. 레거시 캠페인 요소를 포함하며, 그리드펑크(Gridpunk) 세계관 위에서 플레이됩니다.

---

## 게임 개요

| 항목 | 내용 |
|------|------|
| 플레이어 수 | 4인 |
| 장르 | 전략 / 자원 관리 / 구역 장악 |
| 테마 | 그리드펑크 (Gridpunk) — 사이버펑크 변형 |
| 플레이 타임 | 90~150분 (캠페인 1회차) |
| 레거시 요소 | 캠페인 모드 (스티커 · 봉투 해금) |

---

## 세계관 — Ashgrid

Ashgrid는 거대 기업 **Bloc(블록)** 들이 지배하던 도시가 붕괴한 이후의 폐허입니다. Bloc의 잔재 위에서 **Ghost(고스트)** 라 불리는 용병 조직들이 구역을 두고 다투고 있습니다. 플레이어는 각자 하나의 Ghost 조직을 이끌며, Ashgrid의 패권을 차지하기 위해 경쟁합니다.

## 자원 체계

게임에는 4종의 핵심 자원이 존재합니다:

| 자원 | 아이콘 | 설명 |
|------|--------|------|
| **Credit** (자금) | 💰 | 모든 행동의 기본 비용 |
| **Operative** (조직원) | 👥 | 구역 점령 및 방어에 필요 |
| **Intel** (정보) | 🔍 | 적 행동 예측, 이벤트 회피에 사용 |
| **Arms** (무기) | 🔫 | 전투력 증강, 습격에 필수 |

## 라운드 구조

각 라운드는 5단계로 진행됩니다:

### 1단계 — 사건 (Event Phase)
이벤트 카드 1장을 공개합니다. 경찰 단속, 시장 변동, Bloc 개입 등 전체에 영향을 미치는 사건이 발생합니다.

### 2단계 — 계획 (Plan Phase)
모든 플레이어가 **동시에** 행동 카드 2장을 선택합니다 (블라인드). 어떤 행동을 할지 서로 모르는 상태로 계획을 세웁니다.

### 3단계 — 실행 (Execute Phase)
이니셔티브 순서에 따라 행동 카드를 공개하고 해결합니다. 같은 구역을 노린 플레이어끼리 충돌이 발생할 수 있습니다.

### 4단계 — 수익 (Income Phase)
장악한 구역에서 자원을 수령합니다. 구역 유형에 따라 다른 자원을 생산합니다.

### 5단계 — 유지 (Maintenance Phase)
조직원 급여, 거점 유지비를 지불합니다. 자원이 부족하면 조직원 이탈이나 구역 상실이 발생합니다.

## 구역 (Territory) 유형

Ashgrid 맵은 다음 유형의 구역으로 구성됩니다:

| 구역 유형 | 생산 자원 | 특수 효과 |
|-----------|-----------|-----------|
| **마켓** | Credit | 거래 시 수수료 할인 |
| **슬럼** | Operative | 모집 비용 감소 |
| **데이터 허브** | Intel | Veil 돌파 보너스 |
| **무기고** | Arms | 방어 시 전투력 +1 |
| **넥서스** (중앙) | 모든 자원 1씩 | 게임 종료 조건 관련 |

## 승리 조건

- **기본**: 8라운드 종료 시 가장 높은 **지배력 점수(Dominance Score)** 보유
- **즉시 승리**: 넥서스(중앙 구역)를 3라운드 연속 장악
- **비밀 목표**: 각 플레이어의 숨겨진 조건 달성 시 보너스 점수
