# HELIX — 블록 전용 액션카드

**블록**: HELIX  
**전문 분야**: 생체공학·스플라이스·의료 독점  
**주 속성**: ◈B BIO + ◈V VOLT  
**플레이스타일**: 아군 회복 → 스탯 강화 → 생체 교란  
**슬로건**: *"Better by design."*

---

## 덱 구성

블록 플레이어는 **전용 카드 6장 + 공용 카드 3장 = 총 9장**으로 시작한다.

## 플레이 가이드

### 전략 개요

HELIX는 **지속 전투력**으로 승리하는 블록이다.
빠르게 이기지 않는다. 대신 오래 버틴다. 상대가 지칠 때까지.

### 초반 (라운드 1~3)

- **FIELD SPLICE**로 아군 고스트 최대한 보호
- 의료구역 확보 최우선 (◈B 공급원)
- NEURAL OVERRIDE로 경쟁자 속도 저하

### 중반 (라운드 4~7)

- **AUGMENTATION**으로 핵심 아군 고스트 강화
- **QUARANTINE ZONE**으로 구역 요새화
- Ghost 계약 확보 (HELIX 패시브: 고용 비용 −2 활용)

### 후반 (라운드 8~12)

- **HARVEST PROTOCOL** 상시 발동 상태 유지
- **CLONE DECOY**로 결정적 전투 생존
- 주식 방어 + 장기 수익 체계 완성

### 추천 조합

| 조합 | 효과 |
|---|---|
| FIELD SPLICE + AUGMENTATION | 회복 후 강화, 스탯 영구 상승 |
| QUARANTINE ZONE + NEURAL OVERRIDE | 함정 + 행동 차단 = 완벽한 구역 방어 |
| HARVEST PROTOCOL + CLONE DECOY | 분신 전투 후 스플라이스 회수 |

### 주의사항

- **공격력 부족** — 직접 충돌에서 IRONWALL에 밀림
- **VANTA의 DATA WIPE** — HELIX 구역 데이터가 타겟되면 치명적
- **속성 혼합 필요** — ◈B만으로는 조합 효과 제한적
