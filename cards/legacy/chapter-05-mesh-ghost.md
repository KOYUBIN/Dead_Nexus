> 가장 불안한 점: 수신한 사람 모두가 그 문장을 **자신의 옛 이름**으로 읽었다는 것.*

> *다섯 번째로 도시가 깨달은 사실:

## 챕터 개요

메시(The Mesh)가 단순한 통신망에서 **또 하나의 전장**이 된다.  
CIPHER가 먼저 발견하지만, 머잖아 모든 Ghost와 Bloc이 메시의 존재를 인정해야 한다.

- **메시 확장 맵** 공개 (물리 맵과 병행)
- 메시 내 이동·전투 규칙 도입
- AI 의식체 NPC **"SIGNAL"** 등장 — 중립이지만 변덕스러운 강력한 존재
- 물리 공간과 메시 공간의 영향력 상호 교차

이 챕터부터 CIPHER는 단독 전장의 우위를 갖지만, 홀로 남겨질 위험도 생긴다.

## 봉투 E — 내용물

### 1. 메시 확장 맵
- 물리 맵과 동일한 11×11 격자 — **메시 레이어** (투명 오버레이 또는 디지털 맵)
- 각 물리 구역마다 대응 노드
- 노드 간 이동: HACK 판정 또는 MESH 속성 소비
- 베일이 강한 구역은 메시 노드도 차단됨

### 2. SIGNAL — AI 의식체 NPC
**이름**: SIGNAL (자칭 명칭)  
**형태**: 고정된 몸체 없음, 메시 전역에 분산 거주  
**행동 원칙**: 매 라운드 랜덤하게 1개 노드에 "강림" — 해당 노드에 접속한 플레이어와 대화 가능

**SIGNAL 상호작용**:
- **대화 성공 (MESH 판정)**: 퀘스트 힌트 획득, 메시 속성 +3
- **거래 시도**: 데이터 지불 시 레거시 단서, 주가 조작 정보, 엔딩 힌트 제공
- **공격 시도**: 실패하면 Ghost 사이코 +3, 성공 예시는 지금까지 없음

### 3. 새로운 카드 (7장)
- **MESH DIVE** (CIPHER 강화): 메시 내 이동력 2배
- **GHOST PROTOCOL** (CIPHER 강화): 메시 내 이니셔티브 −10
- **SIGNAL WHISPER** (전체 공용): SIGNAL과 접촉 시 1회 무료 대화
- **DEEP DIVE** (Ghost 공용): 메시 안에서만 사용 가능한 전용 공격, ATK +4
- **COLD BOOT** (Bloc 공용): 자사 베일 전역 1라운드 완전 차단 (자사 카드 사용도 제한)
- **MESH BOMB** (CIPHER 한정): 메시 노드 1개 일시 파괴, 해당 물리 구역 통신 마비
- **ASCEND** (레거시): 1회 사용, Ghost가 영구히 메시로 의식 업로드 (몸 포기)

### 4. 메시 전용 이벤트 (3장)
- "SIGNAL BLEED" — 모든 플레이어 사이코 +1
- "MESH WAR" — 두 메시 노드 간 자동 전투 발생 (AI끼리)
- "SIGNAL'S GIFT" — 랜덤 플레이어 1명이 레거시 토큰 2개 획득

### 5. 레거시 스티커
- 📡 "UPLINK" — 메시에 최초 진입한 Ghost 카드에 부착
- 👻 "SIGNAL-TOUCHED" — SIGNAL과 3회 이상 접촉한 캐릭터에 영구 부착
- 🕳️ "MESH SCAR" — 메시 전투 중 HP 0 도달한 캐릭터에 영구 부착

### 6. 스토리 카드 (1장)
*"내가 이 문장을 쓴 적이 있던가? 아니면 이 문장이 나를 쓰고 있는 건가?" — SIGNAL, 추정 발화*

## 챕터 효과

### 영구 변경
1. **메시 맵 영구 공개** — 이후 모든 세션에서 사용
2. **SIGNAL NPC 상시 등장** — 매 라운드 무작위 노드에 강림
3. **메시 침입 권리 보편화** — 챕터 5 이후 모든 Ghost·Bloc이 HACK 판정으로 메시 접속 가능 (단, CIPHER가 여전히 가장 유리)
4. **이중 점유 시스템** — 물리 구역과 메시 노드를 각각 별개로 점유 가능

### 임시 변경 (다음 챕터까지)
- MESH 속성 생산 모든 구역 2배
- 사이코 트랙 상승 속도 +1/챕터
- SIGNAL 접촉 시 보상 2배

## 챕터 5 플레이어 선택

### Bloc 플레이어
**"메시를 활용할 것인가, 차단할 것인가?"**

- **A. 활용**: 자사 전용 메시 노드 2개 영구 획득. 데이터 생산 +2/라운드 영구. 단, 자사 해킹당할 확률 2배
- **B. 차단**: 자사 구역 베일 강도 +2 영구. 자사 메시 노드 전체 봉쇄. 단, MESH 관련 수입 없음 영구

### Ghost 플레이어
**"몸을 남길 것인가, 의식을 올릴 것인가?"**

- **A. 육신 고수**: 메시 안에서는 일반 Ghost. 메시 접속 자원 50% 증가. 보상: 물리 전투 +1 영구
- **B. 의식 업로드 (ASCEND)**: 캐릭터 카드 "GHOSTED" 전환. 몸 포기. 장점: 영구 MESH 자원 2, 메시 내 모든 판정 +2, 사이코시스 면역. 단점: 물리 전투·이동 시 심각한 페널티 — 반드시 메시에서 컨트롤 전환 필요
- **C. 중립 유지**: 특수 보상 없음. 단, 엔딩 분기 시 "불안정한 증인" 자격 획득 — 최종 챕터 특수 선택지 해금
