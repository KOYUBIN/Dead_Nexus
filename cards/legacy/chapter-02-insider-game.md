## 챕터 개요

첫 번째 레이드가 블록의 **외부 방어**가 취약함을 드러냈다면,  
두 번째 균열은 **내부**에서 열린다.

블록은 외부의 고스트를 두려워했지만, 정작 진짜 적은 같은 이사회 테이블에 앉아 있었다.

- 블록 간 상호 지분 매입·스파이전·이사회 공작 본격화
- M&A는 더 이상 경영 용어가 아니라 전쟁 용어가 된다
- 고스트는 블록의 내분을 이용할 기회를 얻는다

이 챕터부터 **금융 전쟁의 시대**가 시작된다.

## 봉투 B — 내용물

### 1. 새로운 구역 타일 (2장)
- **이사회 타워** (금융가 구역에 추가): 해당 블록 M&A 방어 +2, 라운드당 ₵ +2
- **차명 계좌 지점** (유흥가 구역에 추가): 주식 거래 시 공개되지 않음 (익명 거래 가능)

### 2. 새로운 행동 카드 (Bloc 전용 6장)
레거시 해금 카드 — 각 Bloc 덱에 1장씩 + 공용 2장:
- VANTA: **LEVERAGE DOSSIER** — 대상 블록 배신 기록 영구 획득
- IRONWALL: **HOSTILE TAKEOVER** — 지분 매입 시 방어 강제 전투 전환
- HELIX: **POISON PILL** — M&A 시도받으면 자기 주가 절반 삭감, 공격자도 −3
- AXIOM: **FLASH MERGER** — 1라운드 내 지분 +15% 즉시 매입
- CARBON: **LEGACY CLAUSE** — 장기 계약 3개 발동 시 M&A 완전 차단
- 공용 2장: **BOARDROOM LEAK**, **SHELL COMPANY**

### 3. 새로운 Ghost 해금 카드 (2장)
- BROKER: **INSIDER TRADE** — 블록 M&A 선언 시 해당 블록 지분 20% 사전 매입 가능
- CIPHER: **BOARDROOM BACKDOOR** — 이사회 회의 내용 1회 완전 공개

### 4. 새로운 이벤트 카드 (4장)
- "이사회 쿠데타" — 랜덤 블록 수장 교체, 해당 블록 덱 1라운드 셔플
- "지분 공개 의무화" — 모든 블록 현재 상호 지분율 공개
- "주총 소집" — 모든 블록 주가 변동 한 라운드 동결
- "비밀 주주 등장" — 숨겨진 Ghost 1명이 특정 블록 지분 10% 공개

### 5. 레거시 스티커
- 🏷️ "FIRST M&A" — 최초 M&A 선언 블록 카드에 부착
- 🎯 "PREY" — M&A 표적이 된 블록 카드에 부착
- 💼 "BETRAYER" — 이번 챕터에서 동맹을 배신한 블록 카드에 부착

### 6. 스토리 카드 (1장)
*"회의록은 언제나 깨끗하게 남는다. 진짜 결정은 그 옆 방에서 이뤄진다. 우리는 그 옆 방에서 만난다."*

## 챕터 효과

### 영구 변경
1. **배신 기록 시스템** 활성화 — 한 번 맺은 동맹을 깨면 해당 블록 카드에 "BETRAYER" 스티커 영구 부착
2. **M&A 방어 라운드** 추가 — M&A 선언 후 피습 블록은 1라운드의 방어 타임 획득
3. **상호 지분 상한** 도입 — 어떤 블록도 타 블록 지분 50% 초과 불가 (100% 인수는 별도 챕터 6 조건)
4. **주가 변동 이벤트 빈도 +50%** (영구)

### 임시 변경 (다음 챕터까지)
- 모든 Bloc은 라운드당 스파이 토큰 1개 생산
- Ghost는 Bloc 이사회 해킹 시 보상 +2
- 모든 Bloc 간 동맹 카드는 "배신 조항" 자동 삽입

## 챕터 2 플레이어 선택

챕터 종료 시 각 플레이어는 **하나의 선택**을 해야 한다.

### Bloc 플레이어
**"동맹을 택하겠는가, 포식자가 되겠는가?"**

- **A. 동맹**: 다른 Bloc 1곳과 영구 동맹 (상호 M&A 불가, 속성 풀 일부 공유). 단, 배신 시 양측 주가 −10
- **B. 포식자**: 다른 Bloc 1곳을 영구 "표적"으로 지정. 해당 블록에 대한 M&A 시도 비용 −3 (영구)

### Ghost 플레이어
**"중립을 택하겠는가, 내부자가 되겠는가?"**

- **A. 중립**: 어떤 블록과도 지속 계약 체결 금지. 모든 블록 구역 진입 시 추적 저항 +1 (영구)
- **B. 내부자**: 특정 블록 1곳과 영구 계약. 해당 블록 구역 무료 진입. 단, 해당 블록 적과 영구 적대
