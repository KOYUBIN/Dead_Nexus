# 퀘스트 덱 (40장)

**덱 종류**: Quest Deck  
**카드 수**: 40장 (8유형 × 5장)  
**용도**: 맵 이벤트 토큰의 "퀘스트" 공개 시 1장 드로우

---

## 퀘스트 공통 규칙

### 슬롯 & 제한
- 플레이어당 **최대 2개 동시 보유** (BROKER는 3개)
- 슬롯 꽉 차면 새 퀘스트 거부 또는 기존 포기

### 시간 제한
- 각 퀘스트는 **3라운드** 내 완료 필수
- 초과 시 자동 실패 → 렙 −2, 카드 파기

### 의뢰인 유형
| 의뢰인 | 추가 효과 |
|---|---|
| 🏢 블록 | 해당 블록 지분 +5% |
| 🌃 거리 | 렙 +1 추가 |
| 🏚️ 비통제구역 | 희귀 자원 +1 |
| ❓ 익명 | 보상 불확실, 위험 |

## 퀘스트 관련 시스템

### 퀘스트 카드 덱 관리
- 40장 셔플 상태로 시작
- 이벤트 토큰에서 "퀘스트" 공개 시 1장 드로우
- 완료·실패 카드는 폐기 (재사용 안 됨)
- 덱 소진 시 버려진 카드 셔플 후 재사용 (단판 모드)

### 레거시 모드 변화
- 특정 퀘스트는 한 번만 등장 (영구 제거)
- 레거시 퀘스트 추가 (봉투에서 해금)
- 실패한 퀘스트는 캠페인 일지에 기록

### 보상 지급 규칙
- 완료 시점에서 즉시 지급
- 렙·크레딧은 공용 계산, 지분은 해당 블록에만 적용
- 보상이 특정 조건에 묶여 있으면 조건 달성 시 지급
